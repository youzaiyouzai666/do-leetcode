## 环境准备
### vscode中如何直接运行ts文件，只需两步：

 step1：typescript和ts-node的安装

```
npm install -g typescript
npm install -g ts-node
复制代码
```

step2: VScode插件安装 Code Runner

完成以上两步，vscode右上角会有三角运行标志，即可
